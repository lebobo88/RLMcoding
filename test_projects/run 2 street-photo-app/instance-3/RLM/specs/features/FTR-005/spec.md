# FTR-005: Home & Onboarding

## Overview
Compelling home page with hero section and quick onboarding to get users into content fast.

## User Stories
- As a new visitor, I want to understand the app immediately so I decide to use it
- As a first-time user, I want quick onboarding so I can start learning
- As any user, I want easy access to all features from home

## Acceptance Criteria
- [ ] Hero section with dramatic tagline and CTA
- [ ] Feature highlights (3-4 key features with icons)
- [ ] Sample wisdom cards (carousel)
- [ ] Master photographer showcase (horizontal scroll)
- [ ] Clear CTAs: "Start Learning", "Explore Masters", "Take Challenge"
- [ ] Responsive layout (mobile-first)
- [ ] Cinematic entrance animation (aperture opening effect)
- [ ] Footer with links, social, legal
- [ ] Meta tags for SEO
- [ ] Open Graph images for social sharing

## Technical Design
**Route**: `/` (home page)
**Components**: Hero, FeatureHighlights, WisdomCarousel, MasterShowcase, Footer
**Animation**: GSAP entrance timeline (staggered reveals)

## Dependencies
None (landing page)
